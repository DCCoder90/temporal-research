Syntax examples

-- Aggregate
SELECT protocol, COUNT(*) AS n, SUM(bytes) AS total
FROM packets
GROUP BY protocol
ORDER BY total DESC;

-- Filter
SELECT * FROM grpc_calls
WHERE src = 'hello-world-worker'
  AND method LIKE '%Workflow%';

-- Join tables
SELECT p.time, p.src, g.method
FROM packets p
JOIN grpc_calls g ON p.src = g.src
  AND ABS(p.time - g.time) < 0.01
LIMIT 50;

Full SQLite SQL is supported — WHERE, GROUP BY, HAVING, ORDER BY, JOIN, LIMIT, window functions, and all built-in functions. Ctrl+Enter runs the query.
